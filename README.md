> This is an [Odin assignment project](https://www.theodinproject.com/courses/javascript/lessons/testing-practice) to practices Unit Test by using [**Jest**](https://jestjs.io/) testing Framework.

## Introduction

### Tech Stacks
* Jest
* Eslint
* Es6(import/export modules);
* Babel7

### Module to Test
* Capitalize
* Reverse String
* Calculator
* Caesar Cipher
* Array Analysis

## Install

```sh
npm install
```

## Run tests

```sh
npm run test
```
or (continue test)

```sh
npm run watch
```

## syntax check
```sh
npx eslint .
```
or (to fix the error)
```sh
npx eslint . --fix
```
